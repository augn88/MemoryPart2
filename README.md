# **Žaidimas „Atmintis“**

Edukacinis kortų žaidimas, kuriame procedūriniam mokomojo turinio generavimui realiuoju laiku naudojami didieji kalbos modeliai (DKM).

## **Atsisiuntimas ir paleidimas**

Šioje repozitorijoje pateikiami paruošti žaidimo diegimo failai. Pasirinkite savo operacinę sistemą ir sekite žingsnius:

### **Windows**

1. Atsisiųskite .exe failą iš repozitorijos.  
2. Paleiskite jį dukart paspaudę ir galite žaisti.

### **Android**

1. Atsisiųskite .apk failą į savo įrenginį.  
2. Jei reikia, įrenginio nustatymuose leiskite diegti programėles iš nežinomų šaltinių.  
3. Įdiekite failą ir paleiskite programėlę.

### **Linux**

1. Atsisiųskite Linux sistemai skirtą failą.  
2. Suteikite failui vykdymo teises (terminale įveskite: chmod \+x failo\_pavadinimas).  
3. Paleiskite failą (./failo\_pavadinimas).

## **Trumpa instrukcija**

* **Tinklo režimas:** Pradiniame nustatymų lange įveskite savo **Google Gemini API raktą**, kad sistema galėtų realiuoju laiku generuoti naujas edukacines užduotis. Raktas saugomas tik jūsų įrenginyje. API raktą galite įsigyti čia: https://aistudio.google.com/app/api-keys  
* **Lokalus režimas:** Jei API rakto neturite arba nėra interneto ryšio, žaidimas automatiškai naudos iš anksto sugeneruotus atsarginius užduočių rinkinius.
